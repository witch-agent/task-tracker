# Task Tracker

یه ابزار ساده برای مدیریت پروژه‌ها و تسک‌ها

## ویژگی‌ها

- 🗂️ ساختار ۳ سطحی: پروژه → دسته → تسک
- ✨ تیک زدن تسک‌های انجام شده
- 💾 ذخیره خودکار در localStorage
- 🌙 طراحی تاریک و مدرن
- 📱 کاملاً ریسپانسیو

## مثال استفاده

```
📁 برنامه نویسی
   └─ witch agent
       ├─ ☐ نوشتن هدینگ
       └─ ☐ ui جدید

📁 توییت
   ├─ توییت‌های AI
   │   └─ ☐ ایده‌یابی
   └─ توییت‌های کریپتو
       └─ ☐ پست دیفای
```

## استفاده

فایل `index.html` رو باز کنید و شروع کنید به اضافه کردن پروژه‌ها!
